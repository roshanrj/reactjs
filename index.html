Pokemon React App by RoshanRJ

You can access the application by entering '/pokemon' in the browser url bar.

  • Deployment:
    1. First install "gh-pages" by running "npm install gh-pages in the terminal of your project which you need to deployment."
    2. Now link your github repositary to your folder by running "git remote add origin https://github.com/roshanrj/reactjs.git".
    3. Add "homepage":"https://roshanrj.github.io/reactjs" in package.json below "private" property.
    4. Add "predeploy": "npm run build","deploy": "gh-pages -d build" in scripts node of package.json
    5. Run command "npm run deploy" in the terminal. So, your application will get published and can be accessed your the url printed in the terminal.
    6. In my case: https://roshanrj.github.io/reactjs/ or https://roshanrj.github.io/reactjs/pokemon as i m using react-router.
  • Testing:
    1. For Random Pokemon card, cookie will get created once for every day, which will help to generate a random card on daily basis. You can check in the application tab and cookie section. Name of the cookie is "isRandomCardShowed".
    2. Also, created a entry of random pokemon card generated to store the data in localStorage. So every refresh the random card component will be loaded from localstorage.
    3. User can search pokemon by entering the pokemon name in the input box provided.
    4. If the pokemon is found then Pokemon card component will be loaded else it will show "No Pokemon Found!" Message on the page.
